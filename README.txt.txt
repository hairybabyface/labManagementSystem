The application is set up to run an external Tomcat Server. This type of deployment
allows you to run and manage multiple applications (or microservices) on a server
with ease.

INSTRUCTIONS:

Step #1:

Go to C:\Program Files\Apache Software Foundation\Tomcat 10.1\bin and double click on startup.bat.
This should launch the external tomcat server.

Step #2:

Open a web browser and go to http://localhost:8080 (or whatever port Tomcat is running on). 
This should bring up the Apache Tomcat homepage.

Step #3:

Click on Manager App and log in with the following credentials:

Username: admin
Password: admin

Under Applications, click on /app. The application homepage (index) should be displayed.
