How to run the application...

1) Go to C:\Program Files\Apache Software Foundation\Tomcat 10.1\bin
2) Double click on startup.bat
3) Open web browser and go to http://localhost:8080 (or whatever port Tomcat is running on).
   This should show the Apache Tomcat homepage
4) Click on Manager App and log in with the following credentials:
   Username: admin
   Password: admin
5) Under Applications, click on /app
